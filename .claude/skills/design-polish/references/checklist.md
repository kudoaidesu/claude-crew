# 仕上げチェックリスト

実装後、このリストを確認する。

## 視覚的階層

- [ ] 情報の優先度が視覚的に明確
- [ ] セクションが視覚的に分離されている
- [ ] 余白に一貫した意図がある

## インタラクション

- [ ] ホバー/フォーカス/アクティブ状態がある
- [ ] 状態変化にアニメーションがある
- [ ] クリック可能であることが分かる

## アクセシビリティ

- [ ] 44pxタッチターゲット
- [ ] 適切なARIA属性
- [ ] 画像にwidth/height

## 細部

- [ ] 凡例やラベルでUIの意味を説明
- [ ] 空状態のデザインがある
- [ ] ローディング状態がある

## 自問

> 「デザイナーとして納品できるか？」

Noなら、何が足りないかを言語化して修正する。
