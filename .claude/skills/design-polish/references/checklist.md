# 仕上げチェックリスト

実装後、このリストを確認する。

## 視覚的階層

- [ ] 情報の優先度が視覚的に明確
- [ ] セクションが視覚的に分離されている
- [ ] 余白に一貫した意図がある

## インタラクション

- [ ] ホバー/フォーカス/アクティブ状態がある
- [ ] 状態変化にアニメーションがある
- [ ] クリック可能であることが分かる

## アクセシビリティ

- [ ] 44pxタッチターゲット
- [ ] 適切なARIA属性
- [ ] 画像にwidth/height

## 細部

- [ ] 凡例やラベルでUIの意味を説明
- [ ] 空状態のデザインがある
- [ ] ローディング状態がある

## モバイルリストUI（フィールド系）

- [ ] カードタイトルに `overflow:hidden; text-overflow:ellipsis; white-space:nowrap` があるか
- [ ] FABとタブバーに同じアクションが重複していないか
- [ ] 複数ステップ（カメラ→確認→完了）で同じデータの表記形式が統一されているか
- [ ] カード内にネストされたボタンがないか（インラインCTAで代替）
- [ ] 差分/比較表示は「登録前 X% → Y% +Z%↑」のように読み流れる方向か
- [ ] 祝福絵文字は64px以上あるか

## 自問

> 「デザイナーとして納品できるか？」

Noなら、何が足りないかを言語化して修正する。
